# Dashboard ISP

O Dashboard ISP é a plataforma de gestão da ISP Consulte para acompanhamento operacional, analítico e estratégico das rotinas da empresa.

O sistema foi pensado para centralizar informações importantes do dia a dia, oferecendo mais visibilidade sobre tarefas, prazos, produtividade, acompanhamento de horas, usuários, projetos, indicadores e módulos de apoio à operação.

## O que o sistema faz

- acompanha tarefas em andamento, concluídas e atrasadas
- atualiza prazos, status e progresso operacional
- consolida horas trabalhadas e produtividade
- organiza visualização de projetos e responsáveis
- oferece painéis analíticos para tomada de decisão
- apoia gestão de usuários, acessos e áreas do sistema
- reúne módulos complementares usados na rotina da ISP Consulte

## Objetivo

O objetivo do Dashboard ISP é dar mais controle, clareza e agilidade para a gestão da operação, permitindo que a equipe acompanhe entregas, identifique gargalos e mantenha uma visão mais confiável da execução dos projetos.
